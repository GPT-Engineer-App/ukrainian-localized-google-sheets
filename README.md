## Setup

```sh
git clone https://github.com/GPT-Engineer-App/ukrainian-localized-google-sheets.git
cd ukrainian-localized-google-sheets
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
